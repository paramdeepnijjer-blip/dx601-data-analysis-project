# DX601 – Data Analysis Final Project

## Overview
This project was completed as part of the DX601 course and focuses on applying end-to-end data analysis techniques to a real-world dataset. The goal of the project was to explore the data, perform statistical analysis, and communicate insights through visualizations and interpretation.

## Objectives
- Clean and preprocess raw data
- Perform exploratory data analysis (EDA)
- Apply statistical and analytical methods learned in the course
- Interpret results and explain findings clearly

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
- Statistical analysis techniques

## Project Workflow
1. Data loading and inspection  
2. Data cleaning and preprocessing  
3. Exploratory data analysis  
4. Visualization of key patterns and trends  
5. Interpretation of results and conclusions  

## Key Takeaways
- Developed structured approaches to analyzing complex datasets  
- Improved ability to translate raw data into meaningful insights  
- Strengthened data storytelling and interpretation skills

- `ParamFinalProject_DX601.ipynb` – Complete analysis notebook with code, visualizations, and explanations
